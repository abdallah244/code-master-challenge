🎮✨ CodeMaster Challenge ✨🚀

CodeMaster Challenge is an interactive web-based coding quiz game built with Angular.
Learn programming while having fun: solve questions, earn coins 🪙, unlock tools 🛠️, and achieve milestones 🏆!

🎯 Game Concept

The goal is to combine learning & gaming:

✅ Solve coding questions correctly.

🪙 Earn coins for each correct answer.

🔓 Unlock higher difficulty levels.

🛠️ Use tools to help with tricky questions.

🏅 Earn achievements for milestones.

💻 Enjoy dynamic UI and interactive gameplay.

🎮 Features

Difficulty Levels:

👶 Student

🧑‍💻 Junior

💪 Mid-Level

🧠 Expert

Question System:

150 questions per level, 50 displayed randomly.

Randomized order for replayability.

Coin System:

Earn coins for correct answers.

Spend coins on tools or unlock new levels.

Tools:

💡 Hint

✂️ Remove Two Options

⏭️ Skip Question

Achievements:

🏆 Badges for milestones like 100 correct answers or finishing a level quickly.

Bonus Rounds & Multipliers:

Extra coins for streaks of correct answers.

Responsive Design – works on desktop & mobile.

Local Storage – saves progress, coins, and achievements.

🧱 Technologies Used

Angular

TypeScript + SCSS

Angular Material / PrimeNG

LocalStorage / ngx-webstorage

GSAP / Anime.js (animations)

RxJS (timers & reactive behavior)

📄 Pages

🏠 Home Page – Start screen & navigation

🎚️ Levels Page – Select difficulty (Student → Expert)

❓ Quiz Page – Questions, answer choices, progress bar, coin counter

🏪 Shop Page – Buy tools using coins

🏆 Achievements Page – Display earned badges

📊 Result Page – Performance summary after each level

⚙️ Settings Page (optional) – Change theme & sound

💬 Help Page (optional) – Game rules & guidance

🚀 Running Locally
git clone https://github.com/username/CodeMaster-Challenge.git
cd CodeMaster-Challenge
npm install
ng serve


Open your browser at http://localhost:4200/ to play the game.
